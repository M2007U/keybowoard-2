# ‚ú® Features
| features | desc |
|---|---|
| 2D Lattice View | *to show which keys are pressed* |
| Octave Ring View | *to show which pitch is played when mapped to an octave* |
| General Settings | *octave and pitch shifting, wave type, layout type, frequency memory array* |
| Layout Settings | *define how each key gets their frequencies* |
| Layout Explicit | *explicitly set the frequencies of each keys* |
| ASDR settings | *controls how volume changes over time for every keypress* |
| FILTER settings | *decide which spectrum to get muffed* |
| shortcut keys | *available on the website* |
